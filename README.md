# SportAEvent

A full-stack sports event booking application with a React/Vite client and an Express/MongoDB server.

## Structure

- `client/` - frontend application built with React, Tailwind CSS, and Vite
- `server/` - backend API built with Node.js, Express, and MongoDB

## Getting Started

1. Install frontend dependencies:
   ```bash
   cd client
   npm install
   ```
2. Install backend dependencies:
   ```bash
   cd ../server
   npm install
   ```

## Run

- Start client:
  ```bash
  cd client
  npm run dev
  ```
- Start server:
  ```bash
  cd server
  npm start
  ```

## Notes

- Configure environment variables in `server/.env`
- The client and server are separate apps in the same repository
